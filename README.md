#### Instalar

##### macOS

Usando [Homebrew](https://brew.sh/), ejecuta los siguientes comandos:

```bash
# 1. Docker
brew install docker

# 2. Python 3.12
brew install python@3.12

# 3. AWS CLI
brew install awscli

# 4. AWS Tap
brew tap aws/tap

# 5. AWS SAM CLI
brew install aws-sam-cli

# 6. AWS CLI Local
brew install awscli-local
```
